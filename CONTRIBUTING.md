# Contributing

## Pull Request Checklist

- `make test` passes.
- `make lint` passes.
- `make vet` passes.
- User-visible behavior has tests or fixtures.
- Docs are updated when workflow, safety, or output changes.
- No Steam credentials, cookies, API keys, private inventory exports, generated sell reports, or local cache files are committed.
- No auto-listing, auto-selling, or Steam Guard automation is introduced.
